O que sei fazer

Uso linguagens de programação e dados para detalhar histórias interessantes. Também gosto de aprender, nem que seja o mínimo, de novas tecnologias.

Abaixo algumas técnicas e ferramentas que sei usar mas quiser maiores detalhes, dá uma olhada na minha
tragetória profissional:

    Linguagens:

    Python para limpeza, análise e visualização de dados, machine learning básico e web scraping

    SQL para lidar com bancos de dados grandes e complexos

    Javascript e D3.js para visualização de dados

    HTML e CSS básicos para criar páginas bonitas

    Outras PL/SQL, PHP e C#

    Banco de dados:

    Oracle o mais famoso banco de dados do mundo

    MySQL utilizo nas aplicações que faço em PHP

    SQL Server para aplicações desenvolvidas em C#

    SQLite banco de dados pequeno e bem leve, ótimo para aplciações pequenas

    Software:

    Planilhas eletrônicas, seja Excel, Google Sheets ou Open Office
